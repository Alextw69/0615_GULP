• 查詢node -v , 安裝node.js
  • 查詢 npm -v
  • 安裝 gulp , npm install gulp-cli -g , 建立資料夾
  • 終端機 在 bash 之下 , 查詢 gulp -v
  • npm init -y , 產生 package.json 檔 , 紀錄所安裝的各種套件
  • npm install => 快速產生 node_modules
  • http://nodejs.cn/learn/the-package-json-guide 參考指南
  • gulp 版本 有分 3 / 4 , 2種版本 , 互不相容
  • 參考 https://hackmd.io/iaDWtB3NRzmefW2IQltHJw , gulp 4 的安裝筆記次第
  • npm install gulp --save-dev (產生 package-lock.json 及 node_modules) ,要新增 .gitignore(寫入node_modules), 這裡安裝gulp版本是 @ 4.0.2
  • new gulpfile.js檔
  • 終端機上 , 執行指令 gulp do / gulp xxxx
  • gulp --tasks (查詢指令) , 可以看到 執行export 的所有任務(函式)
  • plugins(外掛) , 插件使用
  • npm install --save-dev gulp-concat (合併css) , 產生一支 style.css file
  • npm install --save-dev gulp-concat (合併js) , 產生一支 script.js file
  • npm install gulp-uglify --save-dev (js 壓縮) , 產生一支 minijs file
  • npm install gulp-clean-css --save-dev (css 壓縮) , 產生一支 minicss file
  • npm install gulp-rename --save-dev
